Versão PT-BR 
Projeto de um jogo realizado em Python com o objetivo de demonstrar conhecimentos na linguagem.

O jogo consiste em uma série de 5 labirintos onde o jogador terá que alcançar a saída deles indicada pelo quadrado vermelho no mapa.
Para jogar utilize as setas de direção do teclado, você terá 30 segundos para escapar de cada labirinto. Boa sorte!

Version EN
A game project made in Python with the goal of demonstrating knowledge of the language.

The game consists of a series of 5 labyrinths where the player has to reach the exit indicated by the red square on the map.
To play, use the arrow keys on your keyboard and you will have 30 seconds to escape from each maze. Good luck!
